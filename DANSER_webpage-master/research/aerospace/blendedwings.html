Active Tip Twist for Blended Body Wings

The goal of this project was to develop the technology necessary to create an active twist wing tip for a blended body wing. In order to achieve this, we developed an aeroelastic database which was used with multi-dimensional interpolation to develop an aeroelastic controller which was implemented on an embedded controller and performed benchtop testing.
